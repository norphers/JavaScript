# Introducción.

2015 se lanza ES6 y apartir de ese año los lanzamientos se harán con ciclos 
anuales (ES2016, ES2017, ..., ES2019, ...).

ES5 es soportado por todos los navegadores web actuales.

ES6, 7, 8 sólo son soportados por navegadores de última generación. Para usar el código
en navegadores antiguos, se usan transpiladores a ES5, como Bubble. 
Uno lenguaje que transpila es usar TypeScript.

A partir de 2018 se llaman ESNext.
